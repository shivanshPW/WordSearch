# 🔍 Word Search

A lightweight, client-side Word Search game — built with vanilla HTML, CSS, and JavaScript. Designed primarily for tablet play.

## 🎮 Features

- Find hidden words in a customizable grid
- Multiple difficulty levels (Easy, Medium, Hard)
- Various word categories to choose from
- Adjustable grid size and word count
- Multi-language support (English & German)
- Hint system to help find words
- Victory screen with confetti animation
